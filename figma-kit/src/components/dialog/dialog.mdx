## Close

The button that closes the dialog.

The Close primitive is a slot. Provide it with your own component as children. When no children
are specified, the default icon button will be displayed.

## Header

Header of the dialog. Usually contains a title and some primary controls.

## Title

An accessible title to be announced when the dialog is opened.

## Controls

A container for primary controls shown on the right side of the header. Typically at least
contains a close icon button.
